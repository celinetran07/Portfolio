Live Coding & Sound Performance — 2025

Live Coding Performance

Tools Strudel, Hydra, Flok
Type Live Performance
Year 2025

Thesis

Can code be a musical instrument? This project explores live coding as performance, using Strudel to compose and play music that only exists in the moment it runs. The act of writing and the act of playing happen at the same time, in front of an audience.

Ideation

This was a group live coding performance using Strudel, a browser-based environment built on TidalCycles patterns, chosen for its cycle-based syntax that lets musical structures be built, layered, and destroyed in real time. We worked inside a shared Flok session that unified our Strudel and Hydra code so all performers could write simultaneously in the same environment. The composition moved through distinct sections, from sparse atmospheric openings through rhythmically dense passages into a darker, more dissonant close, built from interlocking patterns of drums, synths, bass, choir samples, and field recordings.

Approach

I composed and performed a structured Strudel arrangement that formed the backbone of the piece. Working from a base I built in advance, I defined the full performance arc using an arrange() function that sequenced each section by duration, while staying free to modify patterns and gain levels live during the performance. My composition included layered drum patterns across multiple sample banks, synthesizer chords and bass lines, a melodic lead section built on a sawtooth synth with FM and filter modulation, choir textures, a darker Vietnamese-scale passage using gm_fx_rain and gm_violin, and a closing organ section. Sound was distributed spatially across the surround system using orbit routing and randomized panning throughout.

Creative Works

The final work is a live performance staged in a surround audio room, where sound could be spatially positioned and moved around the audience in real time. All performers coded simultaneously inside a shared Flok session, with our screens projected to the audience. Live visuals generated in Hydra ran alongside the music, reacting to the performance in real time. The piece moved through several emotional registers, from light and syncopated, through a dense mid-section, into a darker and more ceremonial close. The code stayed visible throughout, making every compositional decision legible to the audience as it happened.

In collaboration with Cathy Tham & Nicolás Moralez.
